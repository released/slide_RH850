Smart Config. – CPOL/CPHA

  • CPOL :
    • 0 : clock idle at LOW
    • 1 : clock idle at HIGH
  • CPHA :
    • 0 : 1st EDGE
    • 1 : 2ND EDGE

CPOL CPHA mode Type
1 1 3 1
0 1 1 2
1 0 2 3
0 0 0 4
